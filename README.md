# Human Face Project (Java)

A simple Java graphics-based project that creates and displays a human face using different shapes, colors, and drawing methods.

## Technologies Used

- Java

## Features

- Draws a human face using graphics
- Uses shapes like circles and lines
- Adds facial features like eyes, nose, and mouth
- Demonstrates Java graphics programming

## Concepts Used

- Java AWT Graphics
- JFrame
- JPanel
- Drawing Methods
- Object-Oriented Programming concepts

## Project Purpose

The purpose of this project is to understand Java graphics programming and learn how to create visual applications using Java.

## How to Run

1. Install Java JDK
2. Compile the Java file:

```bash
javac HumanFace.java
